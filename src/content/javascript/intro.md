# Introduction to JavaScript

Welcome to the JavaScript course! JavaScript is the programming language of the web, making websites interactive and dynamic.

## What is JavaScript?

JavaScript is a **programming language** that enables you to:
- Create dynamically updating content
- Control multimedia
- Animate images
- And much more!

## Your First JavaScript

```javascript
// Display a message
console.log("Hello, World!");

// Create a variable
let name = "Learning Hub";

// Display an alert
alert("Welcome to " + name + "!");
```

## Where JavaScript Runs

| Environment | Description |
|-------------|-------------|
| Browser | Client-side, manipulates DOM |
| Node.js | Server-side JavaScript |
| Mobile | React Native, Ionic |
| Desktop | Electron apps |

## Key Concepts You'll Learn

1. **Variables & Data Types** - Storing information
2. **Functions** - Reusable blocks of code
3. **DOM Manipulation** - Changing HTML/CSS dynamically
4. **Events** - Responding to user actions
5. **Async Programming** - Handling operations that take time
6. **ES6+ Features** - Modern JavaScript syntax

## Why JavaScript?

- **Universal** - Runs everywhere
- **Versatile** - Frontend, backend, mobile, desktop
- **In-demand** - Most popular programming language
- **Community** - Huge ecosystem and resources

> **Remember:** JavaScript is not Java! They're completely different languages.

Let's start coding!
